# Task 4 - Firewall Configuration

## Objective

Configure and test a firewall rule using Windows Defender Firewall.

## Tool

Windows Defender Firewall with Advanced Security

## Procedure

- Opened Windows Defender Firewall.
- Viewed existing inbound rules.
- Created a new inbound rule to block TCP Port 23.
- Verified the rule was successfully added.
- Removed the rule after testing.

## Result

The firewall successfully blocked inbound traffic on TCP Port 23 during testing.

## Conclusion

This task demonstrated how firewall rules control incoming network traffic and help improve system security.
